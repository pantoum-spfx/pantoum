# Upgrade Analyzer

This page is archived background for the optional Claude Code workflow.

If you use the Claude Code plugin, `/pantoum-analyze` can read Pantoum's generated reports and help you inspect the result conversationally. That is useful, but it is not part of the main onboarding path.
